How I work

I am especially interested in problems where the answer is not immediately visible: situations with many variables, different perspectives, imperfect data and a need to explain what is happening in a way that others can actually use.

In practice, I try to understand the context first, then organize the data and use analytics, machine learning or AI only when they help make complexity clearer. The part I enjoy most is connecting the technical work with a narrative that makes the result understandable and useful for decisions.
